TaskBuddyv1.0

Language[s]: JS

Taskbuddy is a task-keeping app I built using electronJS. It was first built very early in my journey and therefore was not very practical, tasks would be reset each time the app was opened. After I had a bit more experience with programming, I took on the task of making it usable and then decided it was now worthy of being on github. Even if nobody else uses it, at least I do.

File In Outv1.0

Language[s]: Lua

File in and out is a very simple and easy to use module for lua that can be installed with luarocks. Fio provides multiple functions for file system operations and includes some basic JSON operations. It was my first experience with uploading modules to a package manager.
